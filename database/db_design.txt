Database provider: SQLite (Room)

Tables:

USERS
- user_id: integer, primary key autoincrement
- login: text, unique
- email: text, unique
- password_hash: text
- theme_preference: integer (boolean)
- language: text
- avatar_uri: text
- created_at: timestamp

HABITS
- habit_id: integer, primary key autoincrement
- user_id: integer, foreign key -> users(user_id)
- title: text
- description: text
- period_n: integer (1, 7, 30)
- current_points: integer (0-60)
- egg_color_hex: text
- last_check_in: timestamp
- hatched_at: timestamp (nullable)

HABIT_LOGS
- log_id: integer, primary key autoincrement
- habit_id: integer, foreign key -> habits(habit_id)
- points_added: integer
- log_timestamp: timestamp

Relations:
users 1 --- N habits
habits 1 --- N habit_logs
